# Intro K8s

10 ejercicios secuenciales para primera vez con Kubernetes.

| # | Carpeta | Objetivo |
|---|---------|----------|
| 1 | 01-pod-basico | Inspección: get, describe, logs, exec |
| 2 | 02-servicio-restart | Service + matar pods → ver restart |
| 3 | 03-fallo-y-logs | CrashLoopBackOff + detectar logs |
| 4 | 04-probes | Liveness + Readiness probes con nginx |
| 5 | 05-rolling-update | Rolling update + rollback de Deployment |
| 6 | 06-configmap | ConfigMap como variables de entorno |
| 7 | 07-sidecar-logs | Logs multi-contenedor / sidecar |
| 8 | 08-describe-debug | kubectl describe como debugging |
| 9 | 09-port-forward | kubectl port-forward |
| 10 | 10-metrics-top | Métricas con kubectl top (metrics-server) |
| 11 | 11-blue-green | Blue/Green a mano: 2 Deployments + Service + selector |
